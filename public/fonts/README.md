# Fonts Directory

Place your custom font files here:

- NotoSansBengali-VariableFont_wdth,wght.woff2 → Bangla font
- Inter-var.woff2 → Latin font (optional)

Download Noto Sans Bengali from:
https://fonts.google.com/noto/specimen/Noto+Sans+Bengali

Or use Google Fonts CDN in app/layout.tsx:
```html
<link href="https://fonts.googleapis.com/css2?family=Noto+Sans+Bengali:wght@400;500;600;700&display=swap" rel="stylesheet" />
```
